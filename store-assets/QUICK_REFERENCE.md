## 🚀 Workflow

1. **Screenshot** (15-30 mins)
   - Open mock pages in Chrome
   - Take screenshots with correct dimensions
   - Save to `screenshots/` folder

2. **Design** (30-60 mins)
   - Use Canva to create promo images
   - Follow brand colors and messages
   - Export as PNG
   - Save to `promo/` folder

3. **Verify** (5 mins)
   - Check dimensions with `sips` command
   - Preview all images
   - Confirm quality

4. **Complete** 
   - Tell me "Materials completed"
   - We move to next phase
