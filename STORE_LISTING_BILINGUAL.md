## 📋 How to Add Both Languages in Chrome Developer Dashboard

### Step 1: Set Primary Language

1. **Find "Language" section**
   - Location: Store listing page
   - Default language: English

2. **Add Chinese**
   - Click "Add language"
   - Select: "Chinese (Simplified)" / "中文（简体）"

### Step 2: Fill in English (Primary)

**Short Description:**
- Copy from "English Version" → "Short Description"
- Paste into "Short description" field

**Detailed Description:**
- Copy from "English Version" → "Detailed Description"
- Paste into "Detailed description" field

### Step 3: Fill in Chinese

**Switch to Chinese tab:**
1. Click "中文（简体）" tab
2. All fields will appear in Chinese

**Short Description:**
- Copy from "中文版本" → "简短描述"
- Paste into "简短描述" field

**Detailed Description:**
- Copy from "中文版本" → "详细描述"
- Paste into "详细描述" field

### Step 4: Save

- Click "Save draft" or "Continue"
- Both language versions will be saved
