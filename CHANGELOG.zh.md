# 更新日志

本文件记录 XCapture 的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.3] - 2026-04-05

### 新增

- 长图切割：自动将长截图按 16:9 比例切割为多张图片（针对屏幕浏览优化）
- 水印：每张切割图片右下角添加 "captured by XCapture Chrome extension" 水印
- 全部下载：一键下载所有切割图���
- 国际化（i18n）：根据浏览器语言自动切换界面（中文浏览器显示中文，其他显示英文），预览模态框内可手动切换语言
- 切割图片复制按钮显示页码信息（如"复制 1/3"）

### 变更

- 切割比例从 A4（1.414）改为 16:9（1.778），屏幕浏览体验更佳

## [1.0.0] - 2026-03-08

### 新增

- 首次发布
- 核心功能
  - 从推文操作栏一键截图
  - 自动为所有推文注入相机按钮
  - 使用 MutationObserver 检测推文
  - 通过 html-to-image 库捕获截图
  - 预览模态框支持复制/下载
- 界面功能
  - 截图时显示加载指示器
  - Toast 通知反馈
  - ESC 键关闭模态框
  - 按钮悬停效果
- 技术细节
  - Manifest V3 支持
  - TypeScript 实现
  - Vite 构建系统
  - 运行时无外部依赖
