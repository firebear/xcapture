# XCapture - X.com 推文截图工具设计文档

## 概述

XCapture 是一个 Chrome 浏览器扩展，用于将 X.com (Twitter) 的推文内容转换为图片。用户可以通过点击推文旁边的按钮，将包含完整信息的推文（作者信息、发布时间、文字内容和多图网格布局）生成为图片，并支持预览、复制到剪贴板或下载到本地。

## 核心需求

- **触发方式**: 在推文操作栏中注入按钮
- **内容范围**: 完整推文信息（作者头像、用户名、发布时间、正文、图片）
- **多图处理**: 保持网格布局
- **输出方式**: 预览后选择（复制到剪贴板或下载）

## 技术方案

### 方案选择：html-to-image

**理由**:
- 现代化库，性能优于 html2canvas
- 对现代 CSS（flexbox、grid）支持更好
- 支持多种输出格式（PNG、JPEG、SVG）
- 社区活跃，持续维护

## 架构设计

### 整体架构

采用 Chrome Extension Manifest V3 标准架构：

- **Content Script**: 注入到 x.com 页面，负责 DOM 操作和用户交互
- **Background Service Worker**: 处理插件生命周期事件（可选）
- **UI 组件**: 预览模态框，显示生成的图片并提供操作选项

## 核心组件

### 1. TweetDetector

**职责**:
- 使用 MutationObserver 监听 DOM 变化
- 识别推文元素（通过 Twitter 的 data 属性或类名）
- 为新加载的推文注入截图按钮

**实现要点**:
- 使用 `[data-testid="tweet"]` 选择器定位推文
- 监听 `article` 元素的子树变化
- 防止重复注入按钮

### 2. ButtonInjector

**职责**:
- 在推文操作栏中插入截图按钮
- 按钮样式与 Twitter 原生风格一致
- 绑定点击事件处理函数

**实现要点**:
- 找到操作栏容器（点赞、转发按钮所在区域）
- 创建按钮元素，添加 SVG 图标
- 注入到操作栏末尾

### 3. ScreenshotCapture

**职责**:
- 克隆目标推文的 DOM 元素
- 清理不必要的元素
- 使用 html-to-image 生成图片
- 处理图片格式和尺寸优化

**实现要点**:
- 克隆 `article` 元素
- 移除交互元素（按钮、链接）
- 内联计算后的样式
- 处理跨域图片资源

### 4. PreviewModal

**职责**:
- 显示生成的图片预览
- 提供复制和下载按钮
- 处理用户交互

**实现要点**:
- 使用 Shadow DOM 隔离样式
- 响应式布局，适配不同屏幕
- 复制到剪贴板使用 Clipboard API
- 下载使用 `chrome.downloads` API 或创建 `<a>` 元素

## 数据流

```
用户点击截图按钮
    ↓
克隆推文 DOM 元素
    ↓
清理和优化 DOM
    ↓
使用 html-to-image 生成图片
    ↓
显示预览模态框
    ↓
用户选择操作
    ↓
执行并关闭模态框
```

## 技术细节

### DOM 克隆策略

- 只克隆推文主体部分（`article` 元素内的核心内容）
- 保留内容：
  - 作者头像和用户名
  - 发布时间
  - 推文文字
  - 图片（保持网格布局）
- 移除内容：
  - 交互按钮（点赞、转发、回复等）
  - 链接悬停效果
  - 推荐和广告

### 样式处理

- 将计算后的样式内联到克隆的 DOM
- 处理 Twitter 的动态 CSS 类名
- 确保图片使用绝对路径
- 设置合适的背景色（白色或用户主题色）
- 添加内边距和圆角

### 图片生成

- 格式：PNG（支持透明背景）
- 质量参数：`pixelRatio = 2`（确保高清晰度）
- 尺寸：自适应推文内容高度
- 跨域处理：使用 CORS 或将图片转为 base64

## 错误处理

### DOM 检测失败
- 显示友好提示："无法识别推文内容，请刷新页面重试"
- 提供手动重试按钮

### 图片生成失败
- 捕获 html-to-image 错误
- 显示错误信息："生成图片失败，请尝试使用浏览器截图"

### 跨域图片问题
- 预先检测图片可访问性
- 失败时使用占位图或跳过该图片

### 性能优化
- 防抖：按钮点击后 1 秒内不可再次点击
- 异步加载：使用 `import()` 动态加载 html-to-image
- 懒加载：仅在需要时初始化库
